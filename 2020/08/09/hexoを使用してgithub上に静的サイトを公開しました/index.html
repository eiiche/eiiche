hexoを使用してgithub上に静的サイトを公開しました

github pagesという静的サイトを公開できる機能がgithub上にあるので、
hexo(https://hexo.io/)
というサイトジェネレータを使用し、github上にサイトを公開しました。

今後はプログラムの勉強で躓いた点などを備忘録的にこちらに記録していきます。

今回hexoを使用してgithub pagesを利用した際に詰まった点・解決法は下記

詰まった点:
github上にdeployしたページを開くとcssが適応されていない
(localサーバ上でページをプレビューした際は問題なし)

解決のために取った手段:
日本語サイト、stackoverflow
chromeデベロッパーツールから見れるファイル構成(chrome上でF12キー押下で閲覧できる画面)

解決した手段:
hexoの設定ファイル(_config.yml)内の編集

URL

If your site is put in a subdirectory, set url as ‘http://yoursite.com/child' and root as ‘/child/‘

url: https://eiiche.github.io/eiiche/
root: /eiiche/
