hexoを使用してgithub上に静的サイトを公開しました

August 9, 2020, 4:24 PM

お勉強の備忘録をどこかに残しておきたいと考えていたところ
github pagesというgithub上に静的サイトを公開できる機能があることを知りました。

ただのhtmlファイルでは外見が寂しいものの、CSSへの理解は薄かったため
hexo
というmarkdownで記事を作成できるサイトジェネレータを使用し、github上にサイトを公開。
しかし情報が古かったり環境が異なったりで中々上手くいきませんでした・・

github上にデプロイしたサイトではなぜかCSSが反映されておらず
(chromeデベロッパーツールで確認したところ、参照先が空になっている)
ハマりかけたものの、
hexoの設定ファイル(_config.yml)内の記述を修正し、CSSの参照先を正しくすることで解決。

markdownに触れるのは初ですが、ブラウザ上でmarkdownが編集可能なサービス
stackedit
を使用して記録中
